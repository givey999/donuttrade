# Section 1: Brief description 

**Section 1: Brief description**

* We are creating a website where people can easily trade in game items from the DonutSMP minecraft server without risking to get scammed.  
* It will only use the in game currency and in game items: spawners..  
* To sign up for this platform, it will need your minecraft username and a password.  
* To sign back into your account, it will be ended both your username and password which were created in the sign up.

# Section 3: Trading page

**Section 3: Trading page**

* The trading page should be available only to signed in users.  
* In the trading page you will see both listings for buying/selling spawners. These spawners will be traded for the money the user has in their account.  
* For example if there is a listing for a user (named user1) selling one spawner for 150$ and another user buys it (named user2), the platform should take the money from user2 account and increase their spawner balance by one and user1 should be credited the 150$.  
* NOTICE: when a user creates a sell listing their spawner balance should decrease instantly, even though no one bought the spawner.  
* NOTICE: when a user creates a buy order, their money balance should decrease instantly, even though no one sold the user spawners.  
* Every buy/sell transaction has a 2% fee that will be taken from the user whose money is deducted ( example: if they place a buy order for 100$, they should be deducted 102$)  
* BEFORE APPROVING A BUY/SELL LISTING, THE PLATFORM SHOULD VERIFY IF THE USER HAS ENOUGH BALANCE FOR THE   
* TRANSACTION AND THE FEE.  
* When buying or selling, users can select how many spawner they want to sell or buy, being limited by the balance in their account.  
* All listings should expire after 24 hours, but on the create listing page they should have an option to pay 10000000 to have their listing last for 48 hours. If a listing is created, the user who created it should not have the option to extend the listing time.
